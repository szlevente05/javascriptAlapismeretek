JavaScript alapismeretek

Introdukció

  • HTML 5 - az oldal tartalma
  • CSS 3 - az oldal elrendezése, stílusa
  • JavaScript - az oldal viselkedése, dinamikája
  • Case sensitive (Kis és nagy betűkre érzékeny)
  • A JavaScript kódot közvetlenül a böngésző hajtja végre, nincs szükség fordítóra.

Adattípusok

  • Aritmetikai adat: egyetlen számot ad vissza
  • String: egyetlen stringet eredményez
  • Boolean (Logikai érték): igaz vagy hamis értéket eredményez
  • Array (Tömb): Több adatot láncszerűen tárol, melyek úgynevezett indexekkel vannak ellátva az elérés érdekében
  • Objektum:
    • Adatszerkezet, mely több adatot is tartalmazhat
    • Függvényeket is tartalmazhat adatok feldolgozására vagy egy adott feladat elvégzésére

Adattípusok deklarálása

  • let x = 69; - Aritmetikai változó (szám)
  • let nev = "Kunu Márió"; - String
  • let vanElete = false; - Boolean (Logikai érték)
  • let megvolt = ["Réka", 10, "Betti", 2, "Bogi", 1, "Zsófi", 0, true, false]; - Array (Tömb)
  • let Sandor = new Ember(96, 176, 21, 70, true); - Objektum

A JavaScript lazán típusos vagy gyengén típusos nyelv. A változók deklarálásánál nem kell megadni az adat típusát.
A program futása során a változónak akár a típusa is megváltozhat!

  • Változó: név, típus, érték.
  • Literál: típus, érték.
  • Deklaráció: A változó elnevezése, hely lefoglalás a memóriában, de nem adunk neki értéket
  • Inicializáció: Kezdőérték adása. Ez az érték a program futása során megváltozhat.

JavaScript változók fajtái:

  • var - Globális szintű változó. Manapság elavult típus, heylette a let kulcsszót használják változók deklarálásánál. Változhat futtatási időben.
  • let - Blokk szintű változó, deklarálási "helyétől" függően nem terjed ki az egész kódra. Változhat futási időben.
  • const - Konstans, vagyis állandó változó deklarálása. Értéke változtathatatlan.

  • let szam = 0;
  • var megegyszam = 69;
  • const harmadikszam = 777 - Soha nem fog megváltozni az értéke!
  • let nevek = ["ertek", 784579, false, true]; - Tömb típusú váltózóknak való értékadás
